# MyFriendBen Grant - PolicyEngine Rules Engine Support

**Foundation**: Impact Charitable / MyFriendBen
**Program**: MyFriendBen Subgrant via Bill and Melinda Gates Foundation
**Amount**: $300,000
**Duration**: 21 months (December 2024 - August 2026)
**Status**: Active
**Fiscal Sponsor**: PSL Foundation

## Overview

This grant supports PolicyEngine's open-source national rules engine as the technical backbone for MyFriendBen's national expansion to three additional states.

## Key Deliverables

- **Rules Engine**: Maintain 99.9% uptime for federal programs (SNAP, WIC, Lifeline, NSLP, SSI, EITC, CTC, Medicaid)
- **State Expansion**: Implement 3 new states (Illinois, Massachusetts, Texas) plus maintain Colorado and North Carolina
- **API Support**: Handle up to 50,000 production queries/month across 5 states
- **Technical Assistance**: 500 person-hours/year for MyFriendBen partners

## Target Impact

- **185,000 households** impacted
- **5 states** served
- **$2B** in total benefit value reported
- **$70M** projected value for households
- **Population**: Individuals and families under 200% FPL
