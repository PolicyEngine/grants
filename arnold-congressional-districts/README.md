# State and Congressional District Database Grant

**Foundation**: Arnold Ventures LLC (Donor-Advised Fund)
**Grant ID**: 24-13103
**Amount**: $273,525
**Duration**: 18 months (June 2025 - December 2026)
**Status**: Active
**Fiscal Sponsor**: PSL Foundation

## Overview

This grant supports PolicyEngine in developing and maintaining a database of state and congressional district level estimates of tax and benefit reforms.

## Key Deliverables

### Phase 1: Development (May - August 2025)

1. **State and District Weight Matrices** (May 31, 2025)
   - Produce first state and district weight matrices
   - Share draft dashboard with Arnold Ventures Public Finance team

2. **State File Calibration** (June 30, 2025)
   - Calibrate all 51 state files

3. **Pilot Walkthrough** (August 15, 2025)
   - Interface and API demonstration
   - Testing with Arnold Ventures Public Finance team

### Phase 2: Launch (September 2025)

4. **Public Launch Event** (September 15, 2025)
   - Live webinar debuting the site
   - Public announcement

5. **Flagship TCJA-Extension Report** (September 15, 2025)
   - Major policy analysis release
   - Demonstrates database capabilities

### Phase 3: Iteration (November 2025)

6. **Second Release Webinar** (November 30, 2025)
   - Updated features and enhancements

7. **Uptake Memo** (November 30, 2025)
   - User status summary
   - Press hits documentation
   - Stakeholder examples

### Phase 4: Closeout (December 2026)

8. **Final Hand-off Report** (December 31, 2026)
   - Dashboard link
   - Documentation

9. **Final Grant Report** (January 31, 2027)
   - Comprehensive project summary
