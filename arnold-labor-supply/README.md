# Labor Supply Elasticities Enhancement Grant

**Foundation**: Laura and John Arnold Foundation
**Grant ID**: 23-10411
**Amount**: $85,200
**Duration**: 7 months (December 2023 - June 2024)
**Status**: Completed
**Fiscal Sponsor**: PSL Foundation

## Overview

This grant supported PolicyEngine by enhancing its microsimulation model to include labor supply elasticities, benefiting researchers, policymakers, and the general public.

## Key Deliverables

### 1. Earnings Elasticities Integration (March 31, 2024)
- Incorporated earnings elasticities into policyengine.org
- Integrated into policyengine-us Python package
- Enabled dynamic analysis of labor supply responses

### 2. Chart Development (April 30, 2024)
- Produced chart depicting earnings changes
- Revised versions of all existing PolicyEngine charts:
  - Budgetary impacts
  - Distributional impacts
  - Poverty impacts

### 3. Technical Report (May 31, 2024)
- Positioned PolicyEngine's implementation in broader labor supply landscape
- Established credibility in economics community
- Promoted adoption among researchers

### 4. Training Materials (June 30, 2024)
- Webinar presentation
- Video tutorial
- Written documentation
- Ensured responsible adoption by policy experts

### 5. Final Grant Report (July 31, 2024)
- Submitted final report to Foundation
